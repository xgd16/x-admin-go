# GoFrame Template For SingleRepo

Quick Start: 
- https://goframe.org/quick

## 项目结构

- `web/` - 后台管理前端（Vue 3 + TypeScript + Vite + Element Plus）
- 后端 - GoFrame HTTP API

## 快速启动

**后端**:
```bash
go run main.go
```

**前端**:
```bash
cd web && npm install && npm run dev
```
